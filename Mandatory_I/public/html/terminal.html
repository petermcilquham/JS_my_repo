Git Kommandoer:

  • git clone [git url link] skaber et lokalt repository ud fra link'et.
  • git pull henter opdateringer ned.
  • git add [file/folder/Or -A for all] tilføjer den fil eller folder man specificerer eller alle filer i den directory man er i til repository.
  • git commit -m "[message]" comitter ændringer med en besked.
  • git push, pusher til github.

Andre terminal kommandoer:

  • cd, change directory, navigerer til et directory
  • ls, list directory, viser en liste over alle elementer i det nuværende directory
  • [program] -v viser hvilken version af programmet der er installeret
  • npm install -g nodemon installerer nodemon globalt
  • npm run nodemon kører nodemon
  • npm run start-dev kører start-dev scriptet i package.json filen
